Neue Template

Neue is a minimalist HTML template for creatives.

Bootstrap 3 was used to build this template.

Download Contents

Unzip your downloaded file and you'll find:

documentation - The files you are reading now.
hammer - The hammer template file.
sketch - Sketch file.
sass - SASS files.
template - All the template files ready to edit and customize.

Contact Form

A fully working contact form is provided, to add you email you'll need to open the file form-process.php.

Go to line 27 and you'll find:

$EmailTo = "email@example.com";

Replace it with your email.
